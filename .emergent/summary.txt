<product_requirements>
The PARAS REWARD app is a crypto-mining platform where users earn PRC (10 PRC = ₹1 INR) via mining, tap games, and referrals. VIP members (₹1000/year) get unlimited PRC validity and full redemption/withdrawal, while free users have 24-hour validity with no redemption. Products are redeemed using a multi-item cart, single secret code, atomic PRC deduction, and 25% cashback, with KYC mandatory for withdrawals. Features include Cashback Wallet (INR, ₹99/month, min ₹10 withdrawal, ₹5 fee), Profit Wallets (for stockist delivery shares/deposits), and a Stock Movement system (Company → Master → Sub → Outlet → Customer traceability). A Security Deposit feature (refundable, 3% monthly return, annual renewal) is implemented. Role-Based Access Control (RBAC) dashboards are crucial, with SendGrid for password recovery, a Support Ticket system, enhanced Admin Stockist/Financial/User/Order Management. Recent updates include explicit PRC expiry for free users, immediate wallet deduction on withdrawal requests, wallet transaction history, delivery charge deduction from user wallets, dynamic contact details, multiple UPI withdrawal options with bank details, QR code display, and a tabbed Admin User Management.
</product_requirements>

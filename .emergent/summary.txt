<product_requirements>
The PARAS REWARD application is a crypto-mining platform where users earn non-transferable PRC (10 PRC = ₹1 INR) through daily mining, a tap game, and referrals. VIP members (₹1000/year, admin-approved) gain unlimited PRC validity and full redeem/withdrawal access, while free users have 24-hour coin validity with no redemption. Product redemption uses a multi-item cart, single secret code per order, atomic PRC deduction, and 25% cashback. KYC is mandatory for withdrawals.

Implemented features include a Cashback Wallet (INR, ₹99 monthly fee, min ₹10 withdrawal, ₹5 fee), Profit Wallets (for Master/Sub/Outlet delivery shares/deposit returns), and a Stock Movement system (Company → Master → Sub → Outlet → Customer with traceability). A Security Deposit feature (refundable deposits with 3% monthly return, annual renewal fees) is in place. Role-Based Access Control (RBAC) dashboards are crucial, with recent additions like SendGrid for profile-based password recovery, a user-facing Support Ticket system, enhanced Admin Stockist and Financial Management, and advanced Admin User/Order Management.
</product_requirements>

<product_requirements>
The PARAS REWARD application is a crypto-mining platform where users earn PRC (10 PRC = ₹1 INR) through various activities. It differentiates users into VIP (unlimited PRC validity, full redemption/withdrawal for ₹1000/year) and free users (24-hour PRC validity, no redemption). Key functionalities include a multi-item cart for product redemption with a single secret code, atomic PRC deduction, and 25% cashback (KYC mandatory for withdrawals). The financial system involves a Cashback Wallet (INR, ₹99/month fee, min ₹10 withdrawal, ₹5 fee), Profit Wallets for stockist shares, and a Stock Movement system. A Security Deposit feature with monthly returns is also implemented. The application requires Role-Based Access Control (RBAC) dashboards for various user types, SendGrid integration for password recovery, and a Support Ticket system. Administrative tools include advanced management for Stockists, Financials, Users, and Orders. Recent feature additions include explicit PRC expiry for free users, immediate wallet deduction on withdrawal, comprehensive wallet transaction history, dynamic contact details, multiple UPI withdrawal options, QR code display, and a tabbed Admin User Management. The user also requested the app to be downloadable as a PWA and a redesigned user dashboard, alongside aesthetic and functional refinements to existing pages like Wallet and Profile.
</product_requirements>

<product_requirements>
The PARAS REWARD application is a crypto-mining platform where users mine PRC (10 PRC = ₹1 INR), with VIP and free users having different PRC validity and redemption rules. Core features include a multi-item cart with atomic PRC deduction, 25% cashback (requiring KYC), a multi-wallet financial system (Cashback, Profit, Security Deposit), and RBAC dashboards. Communication is handled via SendGrid for password recovery and a Support Ticket system. Existing functionalities include PRC expiry, immediate wallet deduction, transaction history, dynamic contact details, UPI withdrawal, QR codes, and tabbed Admin User Management. During this development phase, several major features were added: a full-stack Real-time Notifications system, an Analytics Dashboard, enhanced Pagination and Loading States, Offline Mode (PWA), and Advanced Search & Filters. Additionally, Revenue & Growth features such as a Referral Program Dashboard, a Gamification System, and Flash Sales functionality were implemented.
</product_requirements>

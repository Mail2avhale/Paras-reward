<product_requirements>
The PARAS REWARD app is a crypto-mining platform where users earn non-transferable PRC (10 PRC = ₹1 INR) via daily mining, a tap game (100 PRC/day limit), and referrals (capped at 200). VIP members (₹1000/year, admin-approved) gain unlimited PRC validity and full redeem/withdrawal access, unlike free users (24-hour coin validity, no redemption). Redemption for products uses a multi-item cart, single secret code per order, atomic PRC deduction, and 25% cashback. KYC (Aadhaar, PAN uploads) is mandatory for withdrawals.

Implemented features include a Cashback Wallet (INR, ₹99 monthly fee, min ₹10 withdrawal, ₹5 fee), Profit Wallets (for Master/Sub/Outlet delivery shares/deposit returns), and a Stock Movement system (Company → Master → Sub → Outlet → Customer with traceability). A Security Deposit feature (refundable deposits with 3% monthly return, annual renewal fees) is in place.

Extensive dashboards with Role-Based Access Control (RBAC) are required: Admin (KPIs, user/KYC/VIP/product/order/finance management, settings, reports, audit logs), Master Stockist, Sub Stockist, and Outlet panels. Registration is minimal (email/password), with full profile updates and profile picture upload on a dedicated page. The Home page should be FinanceTech-styled, displaying marketplace products. Recent additions include: SendGrid integration for profile-based password recovery (2-field verification), a user-facing Support Ticket system with admin management, enhanced Admin Stockist and Financial Management (deposits, renewals) with real-time data, and advanced Admin User/Order Management.
</product_requirements>

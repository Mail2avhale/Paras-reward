# PARAS REWARD - Admin Panel

Admin dashboard for PARAS REWARD application.

## Features
- User Management
- KYC Verification  
- Order Management
- Withdrawal Approvals
- Analytics & Reports
- System Settings

## Deployment

### Environment Variables (Set in Vercel)
```
REACT_APP_BACKEND_URL=https://parasreward.com
REACT_APP_APP_NAME=PARAS REWARD ADMIN
```

### Build Command
```
yarn build
```

### Output Directory
```
build
```

## Access
- URL: admin.parasreward.com
- Login: Admin/Manager credentials only

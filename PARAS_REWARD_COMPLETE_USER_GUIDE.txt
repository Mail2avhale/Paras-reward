1. WHAT IS PARAS REWARD?
================================================================================

Paras Reward is a rewards-based platform where users earn PRC (Paras Reward 
Coins) daily through mining. The more you grow your network, the more you 
earn. PRC can be redeemed for real value - gift vouchers, bill payments, 
bank transfers, and more.

Key Concept:
- PRC = Paras Reward Coin (Digital reward currency)
- 1 PRC has a dynamic INR value (changes based on ecosystem health)
- You earn PRC every day through mining
- You spend PRC on real services and products

2. REGISTRATION (SIGN UP)
================================================================================

How to Register:
1. Open the Paras Reward app/website
2. Click "Sign Up"
3. Fill in the required details:
   - Full Name (minimum 2 characters)
   - Email Address (unique - one account per email)
   - Mobile Number (10-digit Indian mobile, unique per account)
   - 6-Digit PIN (your login password - cannot be all same digits like 111111)
   - Referral Code (optional - if someone invited you)

Important Notes:
- Mobile number is REQUIRED and must be a valid 10-digit number
- Each mobile number and email can only be used for ONE account
- If you have a referral code, enter it during registration to be part of 
  someone's network (this benefits both you and the referrer)
- You start as an "Explorer" (free plan) after registration

6. MINING - HOW YOU EARN PRC DAILY
================================================================================

Mining is how you earn PRC every day. Think of it as daily rewards.

How Mining Works:
1. Go to "Rewards" page
2. Click "Start Mining" 
3. Mining session runs for 24 hours
4. Watch your PRC grow in real-time (per second)
5. After session ends (or anytime), click "Collect" to add PRC to your balance
6. A new session auto-starts after collection

Mining Formula:
- Base Mining: 500 PRC/day (your own mining, everyone gets this)
- Network Bonus: Network_Size x PRC_per_user
- Total Daily PRC = (500 + Network Bonus) x Speed Multiplier

PRC Per User Formula (decreasing as network grows):
  PRC_per_user = max(2.5, 5 x (21 - log2(N)) / 14)

  Where N = your effective network size

  Examples:
  +----------------+---------------+
  | Network Size   | PRC per User  |
  +----------------+---------------+
  |       2        |    7.14       |
  |      10        |    6.10       |
  |     100        |    4.62       |
  |    1000        |    3.56       |
  |    4000        |    2.86       |
  |   16384        |    2.50       |
  +----------------+---------------+

Speed Multiplier:
- Explorer: 100% (but CANNOT collect - demo only)
- Elite (Cash/Razorpay): 100% + 30% POPCORN Bonus
- Elite (PRC payment): 100%

Example Calculations:
- Solo user (no network): 500 PRC/day
- 10 active network: 500 + (10 x 6.10) = 561 PRC/day
- 100 active network: 500 + (100 x 4.62) = 962 PRC/day
- 1000 active network: 500 + (1000 x 3.56) = 4,060 PRC/day
- 4000 active network: 500 + (4000 x 2.86) = 11,940 PRC/day

Important: "Active" network members = Elite subscribers who are actively mining.
Explorer users do NOT count in your network.

7. 3-TIER NETWORK CAP - HOW YOUR NETWORK GROWS
================================================================================

Your network doesn't give unlimited bonus. There's a smart 3-Tier cap system 
that rewards team builders:

FORMULA: Network Cap = min(6000, 800 + 16xD + 5xL1)

Where:
  D  = Your Direct Referrals (people YOU invited)
  L1 = L1 Indirect Referrals (people YOUR referrals invited)

+--------+--------------------+------------------+---------------------------+
| TIER   | RANGE              | HOW TO GROW      | DETAILS                   |
+--------+--------------------+------------------+---------------------------+
| Tier 1 | 0 to 800           | Single Leg       | Everyone starts with 800  |
| (Base) |                    | (auto)           | cap. No effort needed.    |
+--------+--------------------+------------------+---------------------------+
| Tier 2 | 800 to 4000        | Direct Referrals | +16 cap per direct        |
|        |                    | (you invite)     | referral you bring.       |
|        |                    |                  | Need 200 directs for max. |
+--------+--------------------+------------------+---------------------------+
| Tier 3 | 4000 to 6000       | L1 Indirect      | +5 cap per L1 indirect.   |
|        |                    | (your referrals  | Need 400 L1 indirects     |
|        |                    |  invite others)  | for max 6000.             |
+--------+--------------------+------------------+---------------------------+

Examples:
+------------------+------------------+---------+
| Direct Referrals | L1 Indirects     | Cap     |
+------------------+------------------+---------+
|        0         |        0         |   800   |
|       10         |        0         |   960   |
|       50         |        0         | 1,600   |
|      100         |        0         | 2,400   |
|      200         |        0         | 4,000   |
|      200         |      100         | 4,500   |
|      200         |      200         | 5,000   |
|      200         |      400         | 6,000   |  (Maximum!)
+------------------+------------------+---------+

What this means:
- Even without inviting anyone, you can benefit from up to 800 network members
- Start inviting friends to unlock Tier 2 (up to 4000 cap)
- When YOUR friends invite THEIR friends, you unlock Tier 3 (up to 6000 cap)
- Maximum network benefit: 6000 active mining members in your tree

13. PRC ECONOMY - HOW PRC VALUE WORKS
================================================================================

PRC has a dynamic value that changes based on ecosystem health.

Current Rate: Check /api/prc-economy/current-rate (shown in app)

The PRC-to-INR rate is determined by 5 ecosystem factors:
1. Total PRC in circulation
2. Active user count
3. Transaction volume
4. Redemption demand
5. Network growth

Important:
- PRC rate is NOT fixed - it's dynamic
- Higher ecosystem activity = better rate
- All prices (subscriptions, services) adjust based on current rate
- The app always shows you the current rate before any transaction

Example:
If PRC rate = 11 (1 INR = 11 PRC):
- Rs.100 voucher = 1,100 PRC base value
- Rs.1,000 bank transfer = 11,000 PRC base value
(+ charges applied on top)

14. CHARGES & FEES EXPLAINED
================================================================================

Every redemption transaction has these components:

1. BASE VALUE: The PRC equivalent of your desired INR amount
   Formula: INR Amount x PRC Rate = Base PRC

2. BURN (Ecosystem Health):
   - Cash/Razorpay Elite: 1% burn
   - PRC Elite: 5% burn
   Purpose: Burns PRC from circulation to maintain value

3. PROCESSING FEE: Rs.10 (converted to PRC at current rate)
   Purpose: Covers transaction processing costs

4. ADMIN CHARGES: 20% of base PRC
   Purpose: Platform operational costs

TOTAL PRC DEDUCTED = Base PRC + Burn + Processing Fee + Admin Charges
YOU RECEIVE = INR equivalent of Base PRC

Example (Cash Elite, PRC Rate = 11):
You want to redeem Rs.1,000:
- Base PRC: 1000 x 11 = 11,000 PRC
- Burn (1%): 110 PRC
- Processing (Rs.10): 110 PRC
- Admin (20%): 2,200 PRC
- TOTAL DEDUCTED: 13,420 PRC
- YOU GET: Rs.1,000

Same example (PRC Elite):
- Base PRC: 11,000 PRC
- Burn (5%): 550 PRC
- Processing: 110 PRC
- Admin (20%): 2,200 PRC
- TOTAL DEDUCTED: 13,860 PRC
- YOU GET: Rs.1,000

15. FAQ - COMMON QUESTIONS
================================================================================

Q: Can I earn PRC without inviting anyone?
A: Yes! You earn 500 PRC/day base mining. But your network cap starts at 800,
   so even without inviting, up to 800 active members below you in the single
   leg tree can boost your earnings.

Q: What is "Single Leg Tree"?
A: All users are arranged in a single line based on joining date. Everyone who
   joins AFTER you is below you in the tree. Your network = active users below
   your position (up to your Network Cap).

Q: What's the difference between Network Size and Network Cap?
A: Network Size = actual active miners below you.
   Network Cap = maximum that count toward your bonus.
   Your effective bonus = min(Network Size, Network Cap).

Q: I'm an Explorer. Can I still mine?
A: Yes! You can start mining and see your earning speed. But you CANNOT collect
   the mined PRC until you upgrade to Elite.

Q: How do I maximize my earnings?
A: 1. Get Elite subscription (Cash for best rates)
   2. Mine daily (start session every 24 hours)
   3. Invite friends (each adds +16 to your cap)
   4. Help your friends invite others (+5 per L1 indirect)
   5. Build to 200 direct + 400 L1 indirects = 6000 max cap

Q: Can I have multiple accounts?
A: No. One account per mobile number and email.

Q: What happens when my Elite subscription expires?
A: You become an Explorer. You can still mine but cannot collect or redeem.
   Your network members also won't count you as active.

Q: Is there a minimum PRC needed to start redeeming?
A: Depends on the service. Gift vouchers start at Rs.100. Bank transfers 
   require minimum Rs.200. Your Redeem Unlock % also limits how much you 
   can redeem.

Q: How long does bank transfer take?
A: After admin approval, typically 1-3 business days.

Q: Can I change from PRC Elite to Cash Elite?
A: When your subscription expires, you can renew with any payment method.

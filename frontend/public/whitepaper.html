🔥 Burn System (Deflation Mechanism)

To maintain PRC value and prevent inflation, we implement automatic burning:

  • Session Burn: PRC burned during active mining sessions
  • Transaction Fee Burn: % of each redemption is burned
  • Inactive User Burn: Gradual reduction for dormant accounts

📈 Active User Factor (AU)

Mining rewards are influenced by Active User metrics:

Formula: Mining Rate = Base Rate × (1 / √Active Users) × Referral Bonus
More active users = distributed rewards. Early adopters benefit more!

🔻 Burn Factor (BF)

The burn rate adjusts dynamically based on:

  • Total PRC in circulation
  • Redemption volume
  • New user growth rate
